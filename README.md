## Steps to Setup

#### Clone The Repository

```bash
  git clone git@github.com:Learn-with-Sumit/batch-3-assignment-1-agency-landing-page-mr-jubayer.git
```

```bash
  cd batch-3-assignment-1-agency-landing-page-mr-jubayer
```

#### Install Dependency

```bash
 npm Install
```

#### Then run in development

```bash
 npm run dev
```
